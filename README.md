# Thilt-Investimentos
Este projeto consiste em uma ferramenta desenvolvida em Excel para simulação de acúmulo de capital ao longo do tempo. O simulador permite visualizar o poder dos juros compostos, diferenciando o valor que saiu do seu bolso (Aporte) do valor gerado pelo tempo (Juros).
Ele foi criado criando uma caixa de título no Excel e digitando o nome Thisoufir
Posteriormente criei linhas com algumas perguntas a serem respondidas
inseri valores (informações) nestas linhas e através da fórnula FV com adaptações F4 e multiplição ajustamos a fórmula 
Logo após criei perguntas para diferentes tempos e ajustamos a formula FV como necessário
Criei uma outra tabela com os tipos de fundos de investimentos FII, perfis de investimentos e percentuais
Abrimos outra aba para chaveamento composto 
Inseri um gráfico para mellhor visualização dos percentuais
